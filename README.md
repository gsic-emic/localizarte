# CHESTv0

Primera versión de [CHEST](https://github.com/pablogz/CHEST) diseñada como herramienta de autoría. Los estudiantes podían utilizar las cosas espaciales y las tareas de aprendizaje que creaban sus profesores haciendo ouso de [Casual Learn](https://casuallearnapp.gsic.uva.es).

Esta aplicación fue diseñada y desarrollada en el marco de la tesis doctoral de [Pablo García Zarza](https://scholar.google.es/citations?user=UlVVjUYAAAAJ) dentro del grupo de investigación [GSIC-EMIC](https://gsic.uva.es).
